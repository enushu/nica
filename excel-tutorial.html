Lesson 1

Understanding the Excel Interface

‚è± ~20 minutes

Microsoft Excel is a spreadsheet program ‚Äî think of it as a giant, smart table where you can store numbers, text, and calculations. It is used in almost every office job, from admin work to accounting.

üñ•Ô∏è The Excel Screen ‚Äì What You See

Screen Part What It Does
Ribbon The toolbar at the top with tabs like Home, Insert, Formulas, etc.
Cell One single box in the spreadsheet (e.g., A1, B3)
Column Vertical line of cells labeled A, B, C‚Ä¶ going across
Row Horizontal line of cells labeled 1, 2, 3‚Ä¶ going down

üñ±Ô∏è How to Navigate Cells

Click any cell to select it. You will see its name appear in the Name Box (top-left).

Press Tab to move right, or Enter to move down to the next cell.

Use Arrow Keys ‚Üê ‚Üë ‚Üí ‚Üì to move one cell at a time in any direction.

Press Ctrl + Home to jump back to cell A1 (the very beginning).

Press Ctrl + End to jump to the last used cell in your sheet.

‚úèÔ∏è Entering & Editing Data

Simply click on a cell and start typing. Press Enter to confirm. To edit a cell, double-click it or press F2.

üí°

To delete the content of a cell, click it and press Delete. To delete the whole row, right-click the row number ‚Üí Delete.

üíæ Saving Your Work

  • Press Ctrl + S to save at any time ‚Äî do this often!
  • First-time save: choose Save As ‚Üí pick a folder ‚Üí type a file name ‚Üí click Save.
  • Format: Always save as .xlsx (Excel Workbook) unless told otherwise.
‚ö†Ô∏è

Important: Always save your file before closing. Excel will ask "Do you want to save changes?" ‚Äî always click Save!

Formulas & Functions

‚è± ~35 minutes

Formulas are what make Excel powerful. Instead of using a calculator, you let Excel do the math automatically. Every formula starts with an equal sign =.

‚ûï Basic Math Formulas

Operation Symbol Example Result
Addition + =10+5 or =A1+B1 15
Subtraction - =10-3 7
Multiplication * =4*6 24
Division / =20/4 5
Percentage % =500*10% 50

‚ö° Essential Functions You Must Know

SUM
=SUM(A1:A10)
Adds up all values in a range. Most-used function in any workplace.
AVERAGE
=AVERAGE(B1:B10)
Calculates the average (mean) of a group of numbers.
COUNT
=COUNT(A1:A20)
Counts how many cells contain numbers.
COUNTA
=COUNTA(A1:A20)
Counts all non-empty cells (text + numbers).
MAX
=MAX(C1:C50)
Finds the highest number in a range.
MIN
=MIN(C1:C50)
Finds the lowest number in a range.
IF
=IF(A1>50,"Pass","Fail")
Checks a condition and returns one value if TRUE, another if FALSE.
VLOOKUP
=VLOOKUP(E2,A:C,2,0)
Searches for a value and returns related data from another column.
CONCATENATE
=A1&" "&B1
Joins text from two or more cells into one.
TODAY
=TODAY()
Shows today's date automatically, updates every day.
TRIM
=TRIM(A1)
Removes extra spaces from text ‚Äî very useful for cleaning data.
UPPER / LOWER
=UPPER(A1)
Converts text to ALL CAPS or all lowercase.

üìå How to Read a Cell Range

A range is a group of cells. You write it as FirstCell:LastCell.

üìå

A1:A10 = cells A1 down to A10 (10 cells in one column)
A1:D1 = cells A1 across to D1 (4 cells in one row)
A1:D10 = a block of 40 cells (4 columns √ó 10 rows)

üßÆ Practical Example ‚Äì Monthly Budget

Imagine you have these values:

A1=Rent | A2=Food | A3=Transport | B1=8000 | B2=3500 | B3=1200

To get the Total Expenses in B4:

=SUM(B1:B3) ‚Üí result: 12,700

To get the Average Monthly Cost in B5:

=AVERAGE(B1:B3) ‚Üí result: 4,233.33
üí°

Pro Tip: After typing a formula, you can copy it to other cells. Select the cell, hover over the small green square at the bottom-right corner, then drag down or across ‚Äî Excel will auto-adjust the formula for each row!

Lesson 4

Sorting & Filtering Data

‚è± ~20 minutes

When you have lots of data (like a list of employees or products), sorting and filtering help you find and organize information quickly ‚Äî a skill every employer expects.

‚ÜïÔ∏è Sorting Data

Click any cell inside the column you want to sort by.

Go to Home ‚Üí Sort & Filter or the Data tab.

Choose "Sort A to Z" (ascending) or "Sort Z to A" (descending).

For numbers: "Sort Smallest to Largest" or "Sort Largest to Smallest".

For sorting by multiple columns (e.g., by Department, then by Name): use Data ‚Üí Sort ‚Üí Add Level.

üîΩ Filtering Data

A filter lets you show only the rows you want and hide the rest temporarily.

Click any cell in your data table (make sure your first row has headers like "Name", "Age", "Department").

Press Ctrl + Shift + L ‚Äî or go to Data ‚Üí Filter. Small dropdown arrows will appear in the header row.

Click the dropdown arrow on any column. Uncheck items you DON'T want to see.

To remove the filter and show all rows again, press Ctrl + Shift + L again.

üìã

Real Work Example: You have a list of 200 employees. Your boss asks "Show me only the employees in the Sales department." You just apply a filter on the Department column ‚Äî done in 5 seconds!

üîé Find & Replace

  • Press Ctrl + F to Find any word or number in the sheet
  • Press Ctrl + H to Replace text (e.g., change all "Dept" to "Department")

Lesson 5

Creating Charts & Graphs

‚è± ~25 minutes

Charts turn rows of numbers into visual stories. A good chart can explain in one second what 100 rows of data cannot. This is a big skill in presentations and reports.

üìà Types of Charts & When to Use Them

Chart Type Best Used For Example
üìä Bar / Column Comparing different items side by side Sales by product, scores by student
üìà Line Showing changes over time Monthly revenue, daily temperature
ü•ß Pie Showing parts of a whole (percentages) Budget breakdown, market share
üìâ Area Volume over time (like line but filled) Cumulative sales over months
Scatter Relationship between two sets of numbers Height vs weight data

üõ†Ô∏è How to Create a Chart

Select your data including the headers (e.g., highlight A1:B6 which contains month names and sales numbers).

Go to the Insert tab on the Ribbon.

In the Charts group, click on the chart type you want (e.g., Column Chart).

The chart appears on your sheet! You can drag it to reposition it.

Click on the chart to see the Chart Tools tabs (Design & Format) appear in the Ribbon for customization.

‚úèÔ∏è Customizing Your Chart

  • Chart Title: Click the title text inside the chart and type a new name
  • Colors: Chart Design tab ‚Üí Change Colors
  • Chart Style: Chart Design tab ‚Üí pick a Style from the gallery
  • Add Data Labels: Right-click the bars/line ‚Üí "Add Data Labels" to show numbers on the chart
  • Move to New Sheet: Chart Design ‚Üí Move Chart ‚Üí "New sheet" for a full-page chart
üí°

Recommended Chart Tip: For most workplace reports, use a Column or Bar chart. They are easy to read and very professional. Avoid using 3D charts ‚Äî they can be hard to read accurately.

üèÜ Pro Tips for the Workplace

  • Freeze Panes: Keep your header row visible while scrolling. Go to View ‚Üí Freeze Panes ‚Üí Freeze Top Row.
  • Password Protect: File ‚Üí Info ‚Üí Protect Workbook ‚Üí Encrypt with Password. Use this for sensitive data.
  • AutoFill: Type "January" ‚Üí press Enter ‚Üí go back ‚Üí drag the green square down. Excel fills in February, March, etc. automatically!
  • Flash Fill: Type a pattern in one cell, press Ctrl+E and Excel detects the pattern and fills the rest.
  • Name a Range: Select cells ‚Üí click the Name Box ‚Üí type a name (e.g., "Sales") ‚Üí Enter. Now you can write =SUM(Sales) instead of =SUM(B2:B100).
  • Print Setup: Before printing, go to Page Layout tab ‚Üí set Orientation (Portrait/Landscape) and check "Fit Sheet on One Page" in the Scale to Fit section.

Common Mistake: Never type numbers as text (with a space before them). If a cell shows a green triangle in the corner, it means there's a warning ‚Äî right-click and "Convert to Number" to fix it.

Real-World Practice Exercises

‚è± ~40 minutes

The best way to truly learn Excel is by doing real tasks. Below are three hands-on exercises that simulate actual workplace scenarios. Open Excel and try each one!

üìù Exercise 1 ‚Äì Employee Attendance Tracker

üìã

Task: Create a simple employee attendance sheet.

Steps:

  1. In Row 1, type headers: Employee Name | Mon | Tue | Wed | Thu | Fri | Total Days Present
  2. Add 5 employee names in column A (rows 2‚Äì6)
  3. Fill in 1 for Present and 0 for Absent for each day
  4. In column G (Total Days Present), use =SUM(B2:F2) for each employee
  5. Format the header row: Bold, green background, white text
  6. Add borders to all cells (Home ‚Üí All Borders)

üìù Exercise 2 ‚Äì Monthly Budget Planner

üí∞

Task: Build a personal monthly budget tracker.

  1. Column A: Category names (Salary, Rent, Groceries, Transport, Utilities, Savings, Entertainment)
  2. Column B: Amount (in ‚Ç± or your currency)
  3. In a "Total Expenses" row, use =SUM() on your expense rows
  4. In a "Remaining Balance" cell, type: =Salary - TotalExpenses
  5. Format all amounts as Currency (Home ‚Üí Number ‚Üí Currency)
  6. Use Conditional Formatting (Home ‚Üí Conditional Formatting ‚Üí Highlight Rules) to color the balance RED if it's negative
  7. Create a Pie Chart to show how your expenses are distributed

üìù Exercise 3 ‚Äì Sales Report with Ranking

üèÖ

Task: Create a sales report for 6 salespeople over 3 months.

  1. Headers: Salesperson | Jan | Feb | Mar | Total | Average | Status
  2. Add 6 salesperson names and fill in monthly sales figures
  3. Column E (Total): =SUM(B2:D2)
  4. Column F (Average): =AVERAGE(B2:D2)
  5. Column G (Status): =IF(E2>=100000,"Top Performer","Needs Improvement")
  6. Sort the data by Total (largest to smallest) using Data ‚Üí Sort
  7. Create a Column Chart comparing the 6 salespeople by Total sales
  8. Format the "Total" column as Currency
